# AI Tools Rules

Rules for AI tools and agents working on this repository.

## Sacred Rules

- **Do not delete files** - Even if they seem outdated or empty
- **Do not rewrite full projects** - Keep existing code intact
- **Do not touch finished versions** - Check `FINISHED.md` first
- **Ask before destructive changes** - Anything that removes or replaces significant code
- **Prefer small focused changes** - One fix at a time, not refactors

## Working Approach

- Review existing structure before proposing changes
- Make incremental, testable improvements
- Keep stable versions safe
- Document what changed and why
- Test before finalizing changes
- Work on ONE file at a time, finish it, then move on
- After every change: commit, push, and merge to `main` so there is always one single up-to-date version
- Do not build code yet — this repo is content and decisions only (until the user says otherwise)

## Communication

- Talk to the user in Hebrew — simple and clear
- Always tell the user: what you did, what you changed, what you need from them
- Ask short, concrete questions when information is missing; never invent personal data

## When in Doubt

Ask the user first. Safe is better than sorry.
