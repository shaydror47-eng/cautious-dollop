## Evaluation Framework

When deciding which idea to build next:

| Factor | Weight | Score |
|--------|--------|-------|
| Earnings impact | 40% | [1-10] |
| Implementation difficulty | 30% | [1-10] |
| Time to build | 20% | [1-10] |
| User value | 10% | [1-10] |
| **TOTAL** | 100% | [Score] |
