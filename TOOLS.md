# Tools

What each tool is used for.

## ChatGPT

General questions, brainstorming, writing, explanations, quick answers.

## Claude Code

Writing code, debugging, refactoring, technical analysis, detailed explanations.

## GitHub Copilot

Inline code completion, quick fixes in editor, context-aware suggestions.

## CMD

Running commands, file operations, system tasks, batch automation.

## ADB

Android phone debugging, app installation, device management, logs and testing.

## Android Phone

Testing apps, work execution, daily usage, real-world validation.

## GitHub

Repository management, version control, issue tracking, backup and history.
